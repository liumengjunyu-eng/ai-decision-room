# AI Decision Room V3 — 图谱 + 辩论双视图

## Objective

将 AI 决策舱（Decision Room）从纯流式辩论升级为「图谱」+「辩论」双视图模式，让用户直观看到 AI 角色的观点空间关系和冲突收敛路径。

## Key Changes

1. **新增「图谱」视图**：6 个 AI 角色以散点节点图排列，按立场（支持/反对/中立）和角色类型定位空间，连线汇聚至 CEO 裁决节点
2. **保留「辩论」视图**：流式打字效果、依次发言、灰→紫闪→绿状态指示保持不变
3. **双视图切换**：顶部切换按钮（图谱/辩论），可随时切换查看
4. **7 个董事会角色**：战略官、批判官、风控官、增长官、洞察官、创新官 + CEO 裁决官
5. **冲突可视化条**：支持 `severity_pct` 比例条（左紫右红）
6. **CEO 结构化决策**：决策 + 置信度 + 理由 + 执行步骤 + 风险标签
7. **免费次数计数器 + 广告解锁**
8. **`/api/run` 真实 API** + Mock 回退
9. **Linear 深色主题**、响应式设计、空态引导
